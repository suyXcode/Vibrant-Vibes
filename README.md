# Vibrant-Vibes

## VibesX Site

Static website for Vibrant Vibes — event-planning and booking site.

## Structure

- `index.html` — home page  
- `about.html` — about us page  
- `events.html` — events listing  
- `css/`, `js/`, `images/` — asset folders  

## How to run locally

Just open `index.html` in your browser (or run a simple static server, e.g. `python3 -m http.server`).  

## Deployment

This repository can be deployed via GitHub Pages or Netlify.
